☁️ Personal Cloud Backup Setup Guide

Is software ka data 100% private rakhne ke liye aapko apne personal Google Drive ka ek link jodna hoga. Bas niche diye gaye 3 chhote steps pure karein:

1 Niche diye gaye Lock-Code ko Copy karein:

function doPost(e) {
  var p = JSON.parse(e.postData.contents);
  var props = PropertiesService.getScriptProperties();
  if (p.action === 'save') {
    props.setProperty('udhar_data', JSON.stringify(p.data));
    props.setProperty('recovered_data', p.recovered);
    return ContentService.createTextOutput("Saved");
  }
}

function doGet(e) {
  var props = PropertiesService.getScriptProperties();
  var d = props.getProperty('udhar_data') || '[]';
  var r = props.getProperty('recovered_data') || '0';
  var out = { data: JSON.parse(d), recovered: Number(r) };
  return ContentService.createTextOutput(JSON.stringify(out)).setMimeType(ContentService.MimeType.JSON);
}

2 Google Script par jakar ise paste karein:

Niche diye gaye button ko daba kar website kholein, wahan "New Project" banakar saara purana code mitayein aur upar se copy kiya hua code paste kar dein.

3 Deploy karke Link yahan daalein:

  • Top Right me Deploy -> New Deployment par click karein.
  • Gear icon (⚙️) par jakar Web App select karein.
  • Who has access ko badal kar "Anyone" karein aur Deploy daba edin.
  • Wahan se milne wale lambe se Web App URL link ko copy karke is App ke upar ⚙️ Settings & Cloud me dakar save kar dein!
